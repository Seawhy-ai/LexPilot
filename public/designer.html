💭 思考过程 ▸
用户询问量子计算的基本概念。我需要用通俗易懂的语言解释，涵盖量子比特、叠加态、纠缠等核心概念，并说明与经典计算的区别。

量子计算是一种利用量子力学原理进行计算的新型计算范式。

与经典计算机不同，量子计算机使用量子比特 (qubit)，它可以同时处于 0 和 1 的叠加态。

核心概念：

  • 叠加态 — 量子比特可同时表示多个状态
  • 量子纠缠 — 两个量子比特间的非局域关联
  • 量子干涉 — 放大正确路径、抵消错误路径
# 简单示例：创建叠加态
from qiskit import QuantumCircuit
qc = QuantumCircuit(1)
qc.h(0)  # Hadamard门 → 叠加态
